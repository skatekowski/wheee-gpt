# wheee-gpt

ChatGPT Custom GPTs für das [WHEEE Protocol](https://github.com/wheee-protocol) – ohne eigene API-Tools. Nutzung über Chat, Copy-Paste von CLI-Ausgaben und optional Copy-Paste-Prototypen.

## Inhalt

| Datei | Beschreibung |
|-------|--------------|
| [Protokoll-GPT-UX-UI.md](Protokoll-GPT-UX-UI.md) | **WHEEE UX/UI-Design Coach** – Mode D, Design-Runden, Prototypen (Copy-Paste-HTML), Design-System, Validierung |

## Verwendung

1. In ChatGPT: **Create a GPT** → Instructions aus der jeweiligen Markdown-Datei kopieren (Abschnitt „Instructions“).
2. Name und Beschreibung aus demselben Dokument übernehmen.
3. Conversation Starters optional eintragen.
4. Keine Actions/API nötig – alles über Chat.

## Prototyp-Option

Der UX/UI-Design Coach kann auf Anfrage **Copy-Paste-HTML-Prototypen** liefern: eine vollständige `.html`-Datei zum Speichern und Öffnen im Browser. Alternativ eine Design-Spec für `wheee prototype` im WHEEE-Projekt.

## Lizenz

Unter gleicher Lizenz wie das WHEEE Protocol (siehe Hauptrepo).
